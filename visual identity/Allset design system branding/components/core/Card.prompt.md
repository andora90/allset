Allset Card — container pour les informations d'étape voyage, KPIs, alertes. `accent` = barre bark top pour les alertes. `dark` = fond Obsidian.

```jsx
<Card variant="light" tag="Vol" title="5 / 8 validés" subtitle="Check-in · Billet · Visa en attente" />
<Card variant="dark" tag="Barcelone · 14 juillet" title="En préparation" subtitle="3 étapes calées · 2 en cours" />
<Card variant="accent" tag="Alerte" title="Autorisation sortie" subtitle="Document enfants requis avant le 7 juillet." />
```

Variants: `light` | `cream` | `dark` | `accent`
Props: `tag` (uppercase label), `title` (Cormorant serif), `subtitle` (DM Sans body)
