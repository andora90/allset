Allset Badge — type de voyage, statut d'étape, catégorie. Toujours uppercase + letter-spacing.

```jsx
<Badge variant="default">Vol</Badge>
<Badge variant="outlined">Hôtel</Badge>
<Badge variant="accent">En cours</Badge>
<Badge variant="light">Location voiture</Badge>
<Badge variant="on-dark">À venir</Badge>
```

Variants: `default` | `outlined` | `accent` | `light` | `on-dark`
