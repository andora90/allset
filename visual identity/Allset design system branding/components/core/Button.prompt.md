Allset Button — use for all interactive actions. `primary` (Obsidian filled) is the default CTA. `on-dark` reverses to Cream fill for Obsidian slide backgrounds.

```jsx
<Button variant="primary">Commencer un voyage</Button>
<Button variant="secondary">En savoir plus</Button>
<Button variant="ghost">Voir le lien partenaire</Button>
<Button variant="accent">Partager maintenant</Button>
<Button variant="on-dark">Action sur fond sombre</Button>
<Button variant="primary" size="sm">Small</Button>
<Button variant="primary" size="lg">Large</Button>
<Button variant="primary" disabled>Désactivé</Button>
```

Variants: `primary` | `secondary` | `ghost` | `accent` | `on-dark`
Sizes: `sm` | `md` | `lg`
Border-radius: always 0 or 2px — never strongly rounded.
